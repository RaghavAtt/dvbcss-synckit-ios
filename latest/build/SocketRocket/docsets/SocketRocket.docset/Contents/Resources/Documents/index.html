SocketRocket Framework

A conforming WebSocket (RFC 6455) client library for iOS, macOS and tvOS. This is Facebook’s SRWebSocket library (made publicly available under a BSD-licence) re-packaged as an iOS framework for convenience. This copy of SocketRocket is only available for use within the SyncKit library.

For the most recent version of SocketRocket, developers should refer to SRWebSocket’s github repo.
